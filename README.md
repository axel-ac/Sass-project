# Sass Website Page Design
## Description
Project aims to create website site using Sass.
## Project Skeleton
```
Sass-website-page-design (folder)
|
|----readme.md                 
|----images             
|----solution
        |----index.html
        |----about.html
        |----contact.html
        |----projects.html 
        |----css (folder)   
              |----main.css  
              |----main.css.map  
        |----scss (folder)   
              |----abstracts (folder) 
                    |----_mixin.scss 
                    |----_variables.scss 
              |----base (folder) 
                    |----_reset.scss               
              |----components (folder) 
                    |----_footer.scss 
                    |----_nav.scss 
                    |----_responsive.scss 
              |----pages (folder) 
                    |----_about.scss 
                    |----_contact.scss 
                    |----_home.scss 
                    |----_projects.scss 
              |----main.scss  
        |----img (folder)
```
